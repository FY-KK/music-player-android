# 音乐播放器安卓版

## 项目简介

一个安卓端音乐播放器应用，支持本地音乐播放和管理。

## 技术栈

- 开发语言：Kotlin
- 构建工具：Gradle
- 音频播放：MediaPlayer / ExoPlayer
- UI框架：Jetpack Compose
- 架构模式：MVVM
